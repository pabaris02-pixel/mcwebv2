# 2D Minecraft в браузере

Небольшой эксперимент, повторяющий базовую механику Minecraft в 2D. Всё работает прямо в браузере: есть процедурно сгенерированный мир, персонаж с физикой и возможностью разрушать/ставить блоки.

## Запуск

Откройте `public/index.html` в любом современном браузере. Отдельный сервер не требуется.

## Управление

- `A` / `D` или стрелки — перемещение влево и вправо.
- `W` или `Space` — прыжок.
- Левая кнопка мыши — разрушить блок.
- Правая кнопка мыши — поставить выбранный блок.
- Колёсико мыши или цифры `1`–`5` — выбор блока в хотбаре.

## Возможности

- Процедурная генерация рельефа с разными слоями блоков.
- Простые деревья с листвой и стволами.
- Камера, следящая за персонажем.
- Физика перемещения, гравитация и столкновения с блоками.
- Хотбар с быстрым переключением блоков.
